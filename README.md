# Overview
The purpose of this application is to give users access to today's headlines and be able to click on an individual article's headline and be taken to that article's details page.

## Setup Instructions
1. Fork the Repo from the [Repository](https://github.com/awirth224/news)
2. Clone the repo
   ```sh
   git clone git@github.com:awirth224/news.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run npm start
   ```sh
   npm start
   ```
5. Use the Link that is given in the Terminal & Open in Browser
   ```sh
   Example: http://localhost:3000/
   ```
